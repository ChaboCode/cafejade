POS Dine In/Take Away (Charges)

Track dine in/take away with charges from point of sale(Restaurant).

Feature :

Manage Takeaway (with charges) /dine in orders .

Search orders with Takeaway order/dine in Backend(Pos Order).

Takeaway/dine printed on order bill receipts.

Overview

In the POS screen, there will be two buttons "Dine in", "Takeaway".

Press 'Dine In' button for selecting dine in option

Press 'Takeaway' button for selecting Takeaway option

Make Payment

Print Receipt

Filter in Pos Order(Takeaway/Dine In)
